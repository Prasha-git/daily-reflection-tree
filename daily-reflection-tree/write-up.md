# Daily Reflection Tree Design

## Question Design
Questions test 3 psychological axes from assignment:
- **Axis 1 (Locus)**: "My preparation" vs "Blame others" reveals internal vs external agency [Rotter 1954]
- **Axis 2 (Orientation)**: "Helped colleague" vs "Expected thanks" shows contribution vs entitlement [Campbell 2004] 
- **Axis 3 (Radius)**: "My problem" vs "Team issue" measures self vs team focus [Maslow self-transcendence]

Each has 4 concrete options forcing spectrum positioning, no vague free text.

## Branching Logic
Decision nodes route by prior answers:
Productive/Mixed → agency-positive path
Tough/Frustrating → agency-challenge path
Reflections interpolate user choices (ex: "You said 'Tough' but also 'Controlled my part'"). Signals tally axis dominance for summary.

## Psychological Sources
- Locus of Control: Rotter (1954) - internal = self-control belief
- Entitlement: Campbell (2004) - expecting unearned rewards
- Self-transcendence: Maslow (1969) - beyond self to team purpose

## Trade-offs Made
- **Fewer deeper questions** (9 vs 30) > shallow coverage
- **Concrete workplace scenarios** > abstract psychology  
- **No moralizing** - reflections guide awareness, don't judge
